Detailed Description

template<uint8_t REQUEST_LIMIT = 1U, typename DispatcherTraits = void>
class middleware::core::ProxyAttributeBase< REQUEST_LIMIT, DispatcherTraits >

Base class for proxy attribute getter functionality. Provides async getter dispatch with future-based request tracking.
