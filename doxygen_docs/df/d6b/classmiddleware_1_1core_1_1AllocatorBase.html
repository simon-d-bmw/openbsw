Detailed Description

template<typename TAllocatorImpl>
class middleware::core::AllocatorBase< TAllocatorImpl >

CRTP base class for memory allocators. Provides thread-safe allocation and deallocation of unique and shared memory buffers. Derived classes must implement allocateImpl, deallocateImpl, regionStartImpl and isPtrValidImpl.

Template Parameters
TAllocatorImplCRTP derived allocator type
