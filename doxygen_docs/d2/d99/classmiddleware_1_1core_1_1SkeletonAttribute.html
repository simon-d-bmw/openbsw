Detailed Description

template<typename T, uint16_t MEMBER_ID, bool AllowsSubscriptions>
class middleware::core::SkeletonAttribute< T, MEMBER_ID, AllowsSubscriptions >

Skeleton-side attribute that stores a value and optionally broadcasts changes to proxies.

Template Parameters
TThe attribute value type
MEMBER_IDThe member ID associated with this attribute
