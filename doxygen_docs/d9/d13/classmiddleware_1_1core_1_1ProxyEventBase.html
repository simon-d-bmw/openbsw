Detailed Description

template<class Proxy, typename EventType>
class middleware::core::ProxyEventBase< Proxy, EventType >

Base class for proxy event subscription handling. Manages registration and invocation of event callbacks on the proxy side.

Template Parameters
ProxyThe concrete proxy class (CRTP friend)
EventTypeThe event data type (or void for notification-only events)
