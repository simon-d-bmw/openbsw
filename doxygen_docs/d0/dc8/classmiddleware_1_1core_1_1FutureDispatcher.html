Detailed Description

template<typename DispatcherTraits, uint8_t REQUEST_LIMIT>
class middleware::core::FutureDispatcher< DispatcherTraits, REQUEST_LIMIT >

Template class managing a pool of futures with callbacks for async request/response.

Template Parameters
DispatcherTraitsCompile-time traits (ArgumentType, METHOD_MEMBER_ID, TIMEOUT_VALUE)
REQUEST_LIMITMaximum number of concurrent in-flight requests
