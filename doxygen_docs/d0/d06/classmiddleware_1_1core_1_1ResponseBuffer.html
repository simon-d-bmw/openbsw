Detailed Description

template<typename Traits, uint16_t RESPONSE_LIMIT>
class middleware::core::ResponseBuffer< Traits, RESPONSE_LIMIT >

Templated response buffer for skeleton responses. Manages a fixed-capacity pool of response slots and handles payload allocation and sending.

Template Parameters
TraitsResponseTraits describing the response type and member ID
RESPONSE_LIMITMaximum number of concurrent pending responses
